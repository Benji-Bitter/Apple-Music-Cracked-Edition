About The Project

Apple Music: Cracked Edition is a lightweight wrapper for the official Apple Music web player. It is designed as a streamlined alternative to the heavier built-in Music app on macOS.

The app works online and gives access to the full Apple Music library while connected, without the resource overhead of the standard desktop client.

Features

  • Lightweight: Minimal resource usage compared to the native app.
  • Native Feel: Wraps music.apple.com in a dedicated macOS window.
  • Media Controls: Play and pause from on-screen controls.
  • Modern Support: Full desktop interface support on Apple Silicon and Intel Macs.

Built With

  • Swift
  • WebKit
  • SwiftUI

Getting Started

Download a release or build locally: Releases

Self-Installation

  1. Clone the repository.
  2. Open Apple Music Cracked Edition.xcodeproj in Xcode.
  3. Configure your signing team in Signing and Capabilities.
  4. Build and run with Cmd + R.

Auto-Update Setup (Optional)

  1. Follow the Sparkle setup guide in SPARKLE_INTEGRATION.md.
  2. Generate EdDSA keys and configure your appcast feed.
  3. Use this flow mainly for distribution maintainers.

Usage

  1. Log in with your Apple ID through the official web login flow.
  2. Browse your library as on the website.
  3. Control playback using in-app controls or the Spacebar shortcut.
